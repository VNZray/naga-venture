## Get started

1. Install dependencies

   ```bash
   npm uninstall -g expo-cli
   npm install -g expo-cli
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

## In the root index, just change the CURRECT_APP to switch between Tourist App, Business Portal, and Tourism Admin
